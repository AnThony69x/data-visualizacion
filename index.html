Sobre el Proyecto

Este proyecto implementa un sistema completo de visualización de datos utilizando un dataset de Spotify. Incluye 10 tipos diferentes de gráficos con personalización avanzada, análisis estadístico y exportación en alta resolución.

Desarrollado como proyecto final de la materia Visualización de Datos en la Universidad ULEAM, demuestra el uso profesional de librerías como Seaborn, Matplotlib, Pandas y Plotly.

Galería de Visualizaciones

1

Personalización Avanzada

Gráfico de barras con control total de ejes, colores y etiquetas. Muestra los top 15 artistas más populares.

2

Mapa de Calor

Matriz de correlación entre variables numéricas para identificar relaciones estadísticas.

3

Histogramas y Densidad

Distribución de frecuencias con curvas de densidad KDE superpuestas.

4

Diagramas de Caja y Bigotes

Análisis de dispersión y detección de valores atípicos (outliers).

5

Gráficos de Densidad KDE

Estimación kernel de densidad univariada y mapas de densidad 2D.

6

Gráfico de Pareto

Aplicación del principio 80/20 para identificar factores clave.

7

Gráfico de Radar

Comparación multidimensional de múltiples métricas simultáneamente.

8

Gráfico de Cascada (Waterfall)

Visualización de cambios acumulativos a lo largo del tiempo.

9

Gráfico de Enjambre (Swarmplot)

Muestra todos los puntos de datos sin superposición.

10

Diagrama de Sankey (Interactivo)

Visualización de flujos entre categorías. ¡Click para ver versión interactiva!

Características del Sistema

Personalización Avanzada

Control total de colores, ejes, etiquetas y estilos visuales.

Menú Interactivo

Interfaz de consola con colores y navegación intuitiva.

Búsqueda de Canciones

Sistema de búsqueda integrado con resultados ordenados.

Comparador de Artistas

Análisis comparativo de múltiples métricas entre artistas.

Alta Resolución

Exportación de gráficos en 300 DPI para uso profesional.

HTML Interactivo

Gráficos Sankey interactivos con Plotly.

Limpieza Automática

Procesamiento y limpieza de datos integrado.

Sistema de Logging

Registro detallado de operaciones con timestamps.

Descargar y Ejecutar

Instrucciones de Instalación

1. Clonar el repositorio

git clone https://github.com/AnThony69x/data-visualizacion.git

2. Instalar dependencias

pip install -r requirements.txt

3. Ejecutar el programa

python main.py
Ver en GitHub

Requisitos del Sistema

  • Python 3.8 o superior
  • pip (gestor de paquetes)
  • 500 MB de espacio libre
  • Windows, macOS o Linux
